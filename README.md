# API

## Article

### Save article

`post`  api/article/save

request body:

- article_id: String
- author_id: String
- title: string
- content_html: string
- content_markdown: string
- is_public: bool
- preview: string
- img_list: []

response body:

- code: integer
  - 0: success
  - 1: error
- errmsg: string
- article_id: String

### Remove article

`post` api/article/remove

request body:

- author_id: String
- article_id: String

response body:

- code: integer
  - 0: success
  - 1: error
- errmsg: string
